Simple metadata editor

PurposeThis sample demonstrates how to create a custom metadata editor and then substitute your editor for the default editor provided with ArcCatalog. It implements the IMetadataEditor interface and is intended to be used within ArcCatalog.

How to use

See Using the samples for help on compiling, setting up the debugger, and running the sample (either an exe or dll).

  1. Open the solution in Visual Studio.
  2. Making sure ArcCatalog is closed, build the solution's class library in Visual Studio. This will register the metadata editor with ArcGIS.
  3. Reopen ArcCatalog. Click the Options item from the Tools menu. This will open the Options dialog.
  4. Select the Metadata tab in the Options dialog. At the bottom of the page there is an Editing Metadata section containing a Metadata Editor combo box. "Simple Editor" will now be available. Choose it and click OK.
  5. Edit the metadata for any catalog item. The custom metadata editor will now appear instead of the FGDC or ISO editors.
